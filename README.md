# Rapidcow Fresh

## How do I install these formulae?

`brew install rapidcow/fresh/<formula>`

Or `brew tap rapidcow/fresh` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "rapidcow/fresh"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
